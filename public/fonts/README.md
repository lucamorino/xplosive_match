Place Roboto Condensed woff2 files here to self-host the font for COEP compatibility.

Recommended filenames:

- RobotoCondensed-Regular.woff2
- RobotoCondensed-Bold.woff2

You can download subsets from Google Fonts, Font Squirrel, or Google Webfonts Helper. When building for production, include only the weights you need to reduce bundle size.
